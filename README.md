## FLICKR PHOTO SEARCH APPLICATION

Ruby (v 2.3.0) on Rails application that finds photos using the Flickr API

### Features
* Sorting
* Multiple Tags
* Masonry Layout
* Facebook Login
* User management
* [Active Admin](http://activeadmin.info/)
* [Bootstrap](http://getbootstrap.com/) Responsive design
